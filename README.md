# node-essential
Minimalistic Nodejs API project setup.

Installation:

1. [nodejs.org](https://nodejs.org)
2. [npmjs.com](https://www.npmjs.com) (includes node)
3. git clone https://github.com/leandroandrade/node-essential
3. cd ./node-essential/
4. npm install
5. npm run dev or npm start
